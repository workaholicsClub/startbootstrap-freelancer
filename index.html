Путь развития
Рабочее название

Онлайн агрегатор образовательных программ и курсов (и, возможно, вакансий). Цель — облегчить поиск подходящих курсов, помочь саморазвитию и трудоустройству. Проект обладает коммерческим потенциалом, полученные средства будут направляться на развитие "Клуба" и некоммерческих проектов

Страницы для проверки гипотез по проекту:

https://vaultn.humanistic.tech/
https://vaultn.humanistic.tech/courses/
https://vaultn.humanistic.tech/vacancy/
https://vaultn.humanistic.tech/vacancy/add.html

Репозиторий проекта на GitHub:

https://github.com/workaholicsClub/competency

Текущий технологический стек:
  • API: PHP7, Spot ORM (м.б. Atlas ORM), Slim Framework, JWT, Guzzle, Composer, Docker, PhpUnit
  • Frontend: JavaScript без фреймворков, WebPack, Hyperscript, JSS, Jest
Проект подходит для желающих:
  • Заниматься продуктовым дизайном
  • Получить навыки проведения пользовательских интервью
  • На практике осваивать методику бережливого стартапа (Lean Startup)
  • Изучать разработку API и консольных приложений на PHP
  • Изучать современный PHP
  • Изучать современный JavaScript
  • Попробовать разработку на ванильном JavaScript с использованием современных библиотек
  • Начать практиковать разработку по методике TDD
  • Освоить работу с базами данных
Основные задачи:
  • Разработка продукта ведется циклами Lean: выдвижение гипотез, разработка минимальной версии для проверки гипотезы, работа с пользователями для проверки гипотезы (пользовательские интервью)
  • Найти целевые аудитории, запустить маркетинг
  • Выработать версию продукта, которая будет проста и понятна для пользователей
  • Разработать рабочий каталог курсов
  • Наполнить каталог данными
  • Прочие текущие задачи

Circles

Коммерческий проект. Дейтинговое приложение. Цель — помочь людям найти себе пару и избежать неприятного опыта. На начальном этапе делается упрощенная веб-версия продукта: онлайн speed-dating с фильтрацией собеседников.

Страница для проверки гипотез по проекту:

https://circles.humanistic.tech/

Репозиторий проекта на GitHub:

https://github.com/workaholicsClub/circles

Технологический стек веб-версии:
  • API: JavaScript, Node.JS, Express, WebSockets, Jest
  • Frontend: JavaScript, React, WebSockets, WebRTC, Jest
Проект подходит для желающих:
  • Изучать разработку API на Node.JS
  • Изучать современный JavaScript
  • Изучать React
  • Изучать работу с видео (WebRTC)
  • Изучать работу с сокетами (WebSockets)
  • Начать практиковать разработку по методике TDD
Основные задачи:
  • Разработка и запуск MVP в веб
  • Разработка приложения (React Native) по итогам запуска MVP
